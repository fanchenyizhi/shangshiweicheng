项目名称：丧尸围城（Unity 3D 游戏）
开发周期：一个半月 | 开发环境：Unity 2022、VS 2022、C#
技术栈：C#、Unity、UGUI、NavMesh、有限状态机 
核心实现：基于面向对象思想封装玩家/丧尸/武器核心类；通过有限状态机实现丧尸追击/攻击行为；完成NavMesh寻路、碰撞检测、UI交互等核心功能
1. 架构设计：采用面向对象思想，封装Player、Monster等核心类
2. 逻辑开发：基于有限状态机（FSM）完成丧尸AI状态切换（巡逻→发现玩家→追击→攻击）；使用NavMesh实现丧尸精准寻路，射线检测完成子弹击中判定；
3. 交互与数据：基于UGUI开发血条、分数、武器切换界面；通过委托/事件实现玩家受伤、击杀丧尸的全局事件通知；使用Json持久化存储各种数据。
项目成果：完成可独立运行的完整游戏Demo，涵盖AI、战斗、UI、数据存储全流程
